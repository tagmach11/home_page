2025 통합 개발 및 운영 기록

2024~2025년 · LX2 개발팀

LX2는 2025년 한 해 동안 플랫폼의 핵심 구조, 보안 체계, 그리고 사용자 경험(UI/UX)을 전면적으로 개편했습니다.

이 기록은 2024년 말 AOF5 업그레이드 착수부터 2025년 말 최신 버전까지의 주요 개발 및 운영 일지를 정리한 공식 뉴스룸 아카이브입니다.

2024년 – 시스템 업그레이드의 시작

10.01

AOF5 업그레이드 착수

정래혁 부장은 LX2의 차세대 개발환경 업그레이드를 공식 발표했습니다. 이번 프로젝트는 JDK·eGovFramework·Spring Framework 등 핵심 기반 기술을 최신 버전으로 이행하여 LX2의 안정성과 보안성을 강화하기 위한 중대한 작업이었습니다.

업그레이드 항목

  • JDK: 1.8 → 17
  • 전자정부프레임워크(eGov): 3.10.0 → 4.3.1
  • Spring Framework: 4.3.25 → 5.3.27
  • Spring Security: 3.0.5 → 5.8.13
  • AOF: 4.3.1 → 5.0.0

LX2는 AOF 핵심 모듈을 완전히 이식하고, 라이브러리 호환성 검증과 커스터마이징을 병행하여 2025년 개편의 기반을 마련했습니다.

2025년 – 플랫폼의 재탄생

01.24

자체 암호모듈 검증 추진

배정훈 대표는 LX2의 보안 경쟁력 강화를 위해 'LX2Crypto V1.0' 자체 암호모듈 개발을 공식 제안했습니다. 이 모듈은 별도의 상용 DB 암호화 솔루션 없이도 시스템 내부에서 암호화를 수행할 수 있도록 설계되었습니다.

03.31

강의실 UI/UX 전면 개편

강인복 차장이 주도한 신규 강의실 화면 리뉴얼이 완료되었습니다. AI 학습보조 기능과 다크테마, 우측 보드형 구조를 도입하여 학습 집중도와 몰입감을 높였습니다.

개편 주요 포인트

  • 강의실 메인: 학습 현황 중심 레이아웃
  • 마이크로러닝 상세: 새로운 목차 구조
  • 학습하기 화면: 반응형·보드형 UI 적용
04.28

콘텐츠 등록/수정 API 개발 완료

김명섭 차장이 주도한 Vimeo·YouTube 자동 연동 API가 개발 완료되었습니다. 콘텐츠 등록 및 수정이 JSON 기반으로 자동화되며, AccessToken을 이용한 보안 구조도 적용되었습니다.

API 예시

  • 등록: /api/lcms/create/vimeo/json.do
  • 수정: /api/lcms/update/youtube/json.do

마무리

LX2는 지난 1년간 "안정성, 보안성, 확장성, 사용자 경험" 이라는 네 가지 핵심가치를 중심으로 시스템을 재정비했습니다.

AOF5 기반 아키텍처 전환, OpenSearch 기반의 검색엔진, 강화된 API와 보안 체계, 그리고 영상·AI 중심의 직관적인 UI까지 — LX2는 기술과 사용자 경험이 조화를 이루는 지능형 학습관리시스템으로 한 단계 도약했습니다.

앞으로도 LX2는 공공기관과 기업교육 시장에서 검증된 품질과 기술력을 바탕으로 지속적인 업데이트와 혁신을 이어가겠습니다.
