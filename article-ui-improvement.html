2025년 5월 · LX2

사용자 경험 향상을 위한 단계적 개선

LX2는 사용자 경험 향상을 위해 홈페이지와 학습관리 화면 전반의 UI를 단계적으로 개선했습니다.

핵심 개선 방향

접근성

쉽고 빠른 정보 접근

편의성

효율적인 운영 환경

일관성

통일된 디자인 경험

홈페이지 하단 구조 개편부터 팝업 정렬 기능, 과정 공유 기능 강화까지 폭넓게 진행되었습니다.

홈페이지 하단(UI Footer) 리뉴얼

2025년 5월 7일 ~ 14일

완전히 새로워진 홈페이지 하단

5월 초부터 중순까지 진행된 리뉴얼을 통해 홈페이지 하단 구조(UI Footer)가 완전히 새로워졌습니다. 기존의 단순 정보 중심 구성에서 벗어나, 사용자 중심의 다양한 기능이 추가되었습니다.

주요 개선 사항

1

'사용방법' 버튼 추가

사용자가 LX2 매뉴얼과 가이드에 쉽게 접근

2

공식 블로그·인스타그램 연동

SNS 콘텐츠와 소식에 빠른 접근

3

하단 배너 디자인 개선

브랜드 정체성과 시각적 통일성 강화

팝업 자동 정렬 기능 요청

2025년 5월 21일

효율적인 팝업 관리

기존에는 관리자 화면에서 팝업을 등록할 때 X, Y 좌표를 직접 입력해야 했습니다. 이로 인해 팝업 위치 관리가 번거로웠던 문제를 해결하기 위해 타일형·계층형 자동 정렬 방식으로 개선 요청이 이루어졌습니다.

기존 방식

  • X, Y 좌표 직접 입력
  • 위치 조정 번거로움
  • 수동 관리 필요

개선 방식

  • 타일형·계층형 자동 배치
  • 등록 순서대로 정렬
  • 효율적인 화면 구성

개선 효과

팝업이 등록 순서와 크기에 맞춰 자동 배치되어 운영자가 보다 효율적으로 화면을 구성할 수 있습니다.

과정 공유하기 기능 개선

2025년 5월 21일

손쉬운 과정 공유

과정 상세 페이지의 공유하기 기능도 함께 개선되었습니다. 기존의 단순 URL·SNS 공유 방식을 보완해, QR코드 자동 생성 기능과 URL 복사 인터페이스 강화가 추가되었습니다.

주요 추가 기능

QR코드 자동 생성

과정별 QR코드를 즉시 생성하여 모바일 공유 및 홍보용으로 활용

URL 복사 강화

원클릭 URL 복사로 다양한 채널에서 빠르게 공유 가능

SNS 공유 최적화

주요 SNS 플랫폼에 맞춘 공유 인터페이스 개선

활용 시나리오

학습자: 과정을 모바일로 쉽게 공유

관리자: 홍보용 QR코드를 손쉽게 생성하여 다양한 채널에서 활용

편리하고 직관적인 학습관리 환경

이번 일련의 UI 개선은 사용자 중심 접근성과 운영 효율성을 동시에 강화한 업데이트로,
LX2가 지속적으로 추구하는 "편리하고 직관적인 학습관리 환경"을 한 단계 발전시켰습니다.
