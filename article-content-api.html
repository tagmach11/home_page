2025년 · LX2

Vimeo·YouTube 자동 연동 API 구축

LX2는 콘텐츠 관리의 효율성을 높이기 위해 Vimeo·YouTube 자동 연동 API 개발을 완료했습니다. 이번 개발을 통해 콘텐츠 등록과 수정 과정이 JSON 기반으로 자동화되어, 관리자는 보다 빠르고 안정적으로 영상 콘텐츠를 관리할 수 있게 되었습니다.

자동화된 콘텐츠 관리 구조

기존 방식의 한계

기존에는 관리자가 콘텐츠를 수동으로 업로드하고, 각 플랫폼별로 개별 설정을 진행해야 했습니다. 이는 시간이 많이 소요되고 실수가 발생할 가능성도 높았습니다.

새로운 자동화 시스템

이번 API 개발을 통해 이러한 과정을 단일 요청(JSON 데이터 전송)으로 처리할 수 있게 되었으며, 여러 개의 콘텐츠를 한 번에 등록하거나 수정할 수 있는 구조로 개선되었습니다.

AccessToken 기반 보안 인증

AccessToken 기반 인증 방식을 도입하여 외부 접근을 차단하고 보안성을 강화했습니다. 이를 통해 시스템은 Vimeo와 YouTube 등 외부 플랫폼과의 연동 과정에서도 데이터 무결성과 접근 제어를 안정적으로 유지할 수 있습니다.

주요 API 구성

Vimeo 콘텐츠 등록 API

/api/lcms/create/vimeo/json.do

Vimeo 플랫폼과 자동 연동되어 콘텐츠 신규 등록을 지원합니다. JSON 형식으로 영상 정보, 메타데이터, 설정값을 전달하면 자동으로 콘텐츠가 등록됩니다.

YouTube 콘텐츠 수정 API

/api/lcms/update/youtube/json.do

YouTube 콘텐츠의 메타데이터 수정을 자동화합니다. 기존 콘텐츠의 제목, 설명, 태그 등을 일괄 수정할 수 있습니다.

대량 콘텐츠 등록 API

/api/lcms/bulk/create/json.do

여러 개의 콘텐츠를 한 번에 등록할 수 있는 대량 처리 API입니다. 시즌별 콘텐츠 업로드나 대규모 콘텐츠 이관 작업에 유용합니다.

효율적인 콘텐츠 관리 환경 구축

이번 API 개발로 LX2는 영상 플랫폼과의 연동을 자동화하고, 관리자의 콘텐츠 관리 부담을 크게 줄였습니다. 앞으로도 LX2는 관리 효율성과 보안성을 동시에 강화하며, 더 나은 콘텐츠 운영 환경을 제공할 것입니다.
