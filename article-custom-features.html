스포츠윤리센터 기능 대응

2025년 10월 27일

운영 효율성과 관리 편의성 향상

스포츠윤리센터의 시스템 요구에 맞춰 검색 기능과 수료증 일괄 출력 기능이 추가되었습니다.

주요 추가 기능

1

고급 검색 기능

과정별, 학습자별 조건에 따라 데이터를 검색하여 효율적인 관리 가능

2

수료증 일괄 출력

수료 대상 학습자의 수료증을 한 번에 출력하여 운영 효율성 향상

관리자 편의성

  • 조건별 검색으로 데이터 관리 효율화
  • 일괄 처리 기능으로 업무 시간 단축
  • 기관별 맞춤 워크플로우 지원

통합검색 퍼블 요청

2025년 10월 28일

기관별 템플릿 일관성 유지

기관별 템플릿 간 기능 일관성을 유지하기 위해 통합검색 기능이 누락된 시안(B, C)에 대한 퍼블리싱 반영 요청이 이루어졌습니다.

개선 목표

  • LX2 전반의 검색 환경 통일
  • 사용자 경험(UI/UX) 개선
  • 템플릿 간 일관성 확보

기대 효과

  • 모든 기관의 통합검색 기능 지원
  • 통일된 검색 인터페이스 제공
  • 사용자 학습 곡선 최소화

OTP 2차 인증 기능 개발 적용

2025년 10월 29일

계정 보안 강화

보안 강화를 위해 OTP(One-Time Password) 2차 인증 기능이 개발서버에 반영되었습니다. 인증 절차는 단계별로 구성되어 있으며, 인증 화면, 설정 화면, 설정 완료 화면이 모두 적용되었습니다.

인증 단계

1

인증 화면

로그인 시 OTP 코드 입력

2

설정 화면

OTP 앱 연동 및 초기 설정

3

설정 완료 화면

인증 설정 완료 확인

보안 강화 효과

사용자: 추가 인증 단계로 계정 보안 강화

기관: 기관 단위의 보안정책 체계적 적용

다양한 기관 환경에 유연하게 대응

이번 신규 기관 맞춤 기능 반영은 LX2가 다양한 기관 환경에 유연하게 대응하기 위한
기술적 확장 과정의 일환입니다.

앞으로도 LX2는 공공기관과 기업 교육 현장에서 요구되는
운영 기능과 보안 요소를 지속적으로 개선해 나갈 예정입니다.
